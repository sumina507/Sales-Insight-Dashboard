# Sales Insight Dashboard Project

## Overview
This project analyzes retail sales data (9,994 records, 2011–2014) to generate insights on product performance, sales trends, and profit drivers.

## Key Deliverables
- Cleaned dataset (CSV and SQLite)
- 6+ visualizations (line charts, bar charts, heatmap, box plots)
- Interactive Streamlit dashboard with filters
- PDF report summarizing findings

## Technologies Used
- Python (Pandas, NumPy, Seaborn, Matplotlib)
- SQLite
- Streamlit
- Plotly

## How to Run the Dashboard
1. Clone this repository
2. Install requirements: `pip install streamlit pandas plotly sqlite3`
3. Run: `streamlit run app.py`

## Key Insights
- Seasonal peaks in September and December
- Discount has a negative correlation with profit (-0.22)
- Technology category generates highest sales and profit
- South region shows highest profit variability
